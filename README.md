# Windows-8.1-Activator
Activate your Windows 8.1 for free. 100% safe and working.
Step-1: Copy the code from the main file "Activator".
Step-2: Open notepad (Notepad++ is recommended) and paste the code.
Step-3: Save it with the extension ".bat" or".cmd" (Code won't work unless saved with any of these extensions).
Step-4: After saving, the file will be saved as an "Command prompt file". Run the file as Administrator (make sure to have internet connection).
Step-5: Wait for a few seconds and it will activate your copy of windows 8.1 for free.
